Manual Installation Guide

Numba provides 3-5x speedup through Just-In-Time compilation:

pip install numba

Works on any system (CPU acceleration)

CuPy provides 10-50x speedup with NVIDIA GPUs:

1. Check CUDA version:

nvcc --version

2. Install CuPy for your CUDA version:

# For CUDA 11.x
pip install cupy-cuda11x

# For CUDA 12.x
pip install cupy-cuda12x

Requirements:

  • NVIDIA GPU with CUDA support
  • CUDA Toolkit installed
  • Compatible GPU drivers

Common Issues:

  • CUDA not found: Install CUDA Toolkit from NVIDIA
  • CuPy installation fails: Check CUDA version compatibility
  • GPU not detected: Update GPU drivers
  • Memory errors: Reduce batch size or use CPU fallback

Check GPU availability:

python -c "import cupy; print('CuPy works!')"
Note: After installing GPU acceleration packages, restart the application to see the changes. The system will automatically detect and use the best available acceleration method.
