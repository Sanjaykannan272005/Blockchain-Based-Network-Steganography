Usage Instructions
Sending Messages
  1. Enter your secret message
  2. Set a strong password
  3. Specify target IP address
  4. Select steganography methods
  5. Click "Send Message"
Receiving Messages
  1. Enter the same password used for encryption
  2. Set network interface (usually "any")
  3. Set capture duration
  4. Select the same methods used for sending
  5. Click "Start Capture"
Important: This tool requires administrator/root privileges for packet injection and capture. Make sure you have the necessary permissions and are using it in compliance with local laws.
